📚 Как работает App Free Builder Pro

🎯 Основная идея

Это визуальный конструктор Android-приложений. Вы собираете экран из готовых компонентов, настраиваете их внешний вид и поведение, а затем скачиваете готовый XML-файл для Android Studio.

🖱️ Drag & Drop компонентов

  • Добавление: Хватаете компонент слева и перетаскиваете в телефон. Он появится внизу списка.
  • Перемещение внутри телефона: Зажмите компонент мышкой и тяните вверх или вниз — он поменяется местами с соседними. Зелёная линия показывает, куда встанет элемент.
  • Клик по компоненту: Выделяет его и открывает редактор свойств справа.

⚙️ Редактор свойств (справа)

  • Меняйте текст, цвет, размер, шрифт и другие параметры.
  • Для цветов есть удобный color picker — просто кликните на квадратик.
  • Все изменения видны сразу в телефоне.

⚡ Функции компонентов

  • У каждого компонента есть раздел "Функции" в редакторе.
  • Кнопка: onClick (действие при клике), onLongClick, navigateTo (переход на экран)
  • Поле ввода: onTextChanged (реакция на ввод), onSubmit
  • Переключатель: onCheckedChanged (реакция на переключение)
  • Слайдер: onValueChanged (реакция на изменение)
  • Картинка/Текст/Карточка: onClick (реагируют на нажатие)
  • Нажмите на чипсы с названиями функций, чтобы включить/выключить их.

💾 Сохранение

  • Проект автоматически сохраняется в браузере (localStorage).
  • Кнопка 💾 Сохранить — принудительное сохранение.
  • Кнопка 📂 Загрузить — восстановить последнее сохранение.

📦 Скачивание

  • Кнопка 📦 Скачать генерирует XML-файл разметки Android.
  • Его можно сразу скопировать в проект Android Studio в res/layout/activity_main.xml.

⌨️ Горячие клавиши

  • Delete — удалить выделенный компонент
  • Escape — снять выделение
